# Public Assets

Save the following files in this folder:

- `logo.png` — the full **PRODUCT WITH ANDREW** wordmark (used in navbar & footer)
- `logo-square.png` — 1024×1024 square version for OG image / favicon (optional)
- `og-image.png` — 1200×630 social sharing image (optional, falls back to logo-square.png)

The site references `/logo.png` and `/logo-square.png` directly.
